## 你遇到的问题

你试过的每一个效率工具，都在让你做**更多决定**：用哪个清单？看哪个视图？下一步做什么？拖动、重排、再规划、再排优先级。对一个 ADHD 大脑来说，下午两点的这点摩擦不是功能——而恰恰是一天悄悄崩掉的地方。

你不需要再多一个「整理」生活的地方。你需要的，是有个东西直接告诉你**现在**该做什么——而且表针已经在走了。

## 它的答案

Don't Think Mode 只让你看见**一件事**：现在该做什么，还剩多少时间。

整个主屏就这些。没有收件箱，没有待办积压，没有「我到底从哪开始」。

而当你踏进一个专注时间块的那一刻——**计时器已经在走了。** 没有「开始」按钮。那种「时间在动，不管你动不动」的感觉？这不是 bug，这就是整个产品本身。

## 为什么它对 ADHD 大脑有效

- ⏱️ **计时器永不停。** 进入 Deep Work 时间块，专注番茄钟自动开始。不用决定、不用预热、不用「我五分钟后再开始」。「动起来」是默认状态。
- 🎯 **一个屏幕，一个答案。** 主屏只回答一个问题——*现在做什么*——一眼看完。其余一切都只在一次点击之外，需要时才出现。
- 🤖 **AI 帮你排序，你不用。** 把今天的任务原样丢进去，它会结合你的目标自动排序——本地排，或用你自己的 OpenAI / Gemini key。你自上而下执行，再也不为「先做哪个」纠结。
- 📈 **会变成行动的分数。** 不是让你愧疚的连续打卡，而是告诉你今天还剩下哪些**具体**的高价值动作——把「再努力点」变成「做这个」。
- ↩️ **没做完的自动顺延。** 今天没关掉的任务自动带到明天，并重新排序。不用收拾残局，不用陷入自责循环。
- 🔒 **100% 本地。无账号、无云端、无追踪。** 一切都存在你的浏览器里。换设备就导出一份 JSON 备份。你的一天，不是任何人的数据。

## 设计原则

1. **砍掉选择，而不是砍掉功能。** 每个屏幕都必须一眼回答*「现在做什么」*。
2. **让时间可见。** 倒计时优先于清单，势头优先于规划。
3. **永远本地优先。** 没有登录墙，没有服务器往返，没有埋点。

## 快速开始

```bash
npm install
npm run dev      # → http://localhost:4318
```

然后在手机上打开，**添加到主屏幕**——它会作为一个独立、可离线的 App 安装。

### 可选：用真正的 LLM 排序任务

内置的本地启发式排序器零配置即用。想用 LLM？把一个 OpenAI（`sk-…`）或 Google
Gemini（`AIza…`）的 key 粘到 **设置 → AI 排序** 里。key 只存在你的浏览器，只用来
给任务排序——永远不会离开你的设备。

## 让它成为你的

种子目标和日程都是中性的默认值，在 App 里就能改：

- **目标** —— 你真正的优先级（排序器会据此给任务加权）。
- **日程** —— 你的时间块、深度工作窗口、起床 / 睡觉时间。
- **设置** —— 番茄钟时长、显示称呼、通知。

你输入的任何东西都不会离开你的设备。

## 技术栈

- **Next.js 14**（App Router）· **TypeScript** · **Tailwind CSS**
- **Zustand** —— 持久化、本地优先的状态管理
